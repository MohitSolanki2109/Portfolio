  Problem: "Page looks broken"
  Solution: 1. Ctrl+Shift+Delete (clear cache)
            2. Hard refresh: Ctrl+Shift+R
            3. Try different browser

  Problem: "Images not loading"
  Solution: 1. Check image URL is correct
            2. Verify CORS if external
            3. Use relative paths for local images
            4. Use absolute URLs for web images

  Problem: "Theme doesn't save"
  Solution: 1. Clear localStorage
            2. Check cookies enabled
            3. Use private/incognito window

  Problem: "Mobile menu doesn't work"
  Solution: 1. Check JavaScript is enabled
            2. Try different mobile browser
            3. Check viewport meta tag

  Problem: "Form not working"
  Solution: 1. Connect to backend (see above)
            2. Check console for JavaScript errors
            3. Try different form service
